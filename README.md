# Telangana Weather Data Analysis (Power BI)

##  Overview
This Power BI project analyzes weather patterns across Telangana using historical meteorological data.  
The goal is to visualize trends, understand climate variations, and identify patterns in temperature, humidity,rainfall and, wind across different districts of Telangana.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Objectives
- Perform in-depth analysis of weather parameters across Telangana.
- Visualize temperature, humidity, wind and rainfall trends over time.
- Identify seasonal variations and district-wise weather behavior.
- Build an interactive and insightful Power BI dashboard.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Dataset Information
The dataset used for this project contains weather observations collected from various districts in Telangana.

**Key Attributes:**
- `Date`
- `District`
- `Temperature (°C)`
- `Humidity (%)`
- `Rainfall (mm)`
- `Wind Speed (km/h)`

> The data was cleaned and structured before visualization to ensure accurate insights.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Insights & Findings
- Average temperature and humidity patterns vary significantly between northern and southern Telangana.  
- Monsoon months (June–September) show a sharp increase in rainfall and humidity.  
- Dry regions experience higher temperature fluctuations year-round.  
- Visualization reveals correlation between temperature and humidity trends across districts.  

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Dashboard Features
This Power BI report includes:
- **Interactive Filters** — for date, district, and weather metrics.  
- **Trend Analysis** — temperature, rainfall, and humidity over time.  
- **Comparative Charts** — district-wise and seasonal comparisons.  
- **KPIs & Cards** — average temperature, total rainfall, and humidity metrics.
